# AI Assessment Assistant

A simple AI summarization prototype built with Next.js, TypeScript, and shadcn/ui components.

## Features

- Clean, modern UI matching the design specification
- Blue banner header with AI Assessment Assistant branding
- Generate with AI button (mock functionality - no backend required)
- Collapsible Summary Notes and Annotations section
- Three text areas for:
  - Risk Mitigation Commentary
  - Challenge themes
  - Overall Assessment Results Summary
- Info box with instructions

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app directory with pages and layout
- `/components/ui` - shadcn/ui components (Button, Textarea, Accordion, Card)
- `/lib` - Utility functions

## Technologies

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Lucide React (icons)
